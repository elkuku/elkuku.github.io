# IITC Plugins

A collection of plugins for [IITC](https://iitc.app).

* [KInventory](https://github.com/elkuku/iitc-kuku-inventory) Display the agents inventory. [Download](https://github.com/elkuku/iitc-kuku-inventory/releases/latest)
* [KExport](https://github.com/elkuku/iitc-kuku-export) Export portal data including state and key info. [Download](https://github.com/elkuku/iitc-kuku-export/releases/latest)
* [KZoomToPortal](https://github.com/elkuku/iitc-kuku-set-zoom-portals) Set the zoom level to "portal". [Download](https://github.com/elkuku/iitc-kuku-set-zoom-portals/releases/latest)

### Helper Plugins

These plugins are meant to be used by other plugins.

* [Helper Handlebars](https://github.com/elkuku/iitc-kuku-helper-handlebars) Integrates the [Handlebars](https://handlebarsjs.com/) template engine. [Download](https://github.com/elkuku/iitc-kuku-helper-handlebars/releases/latest)
* [Helper Inventory]() Fetch and parse the Ingress inventory.
